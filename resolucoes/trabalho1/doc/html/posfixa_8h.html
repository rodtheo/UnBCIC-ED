Descrição detalhada

Arquivo contendo as assinaturas das funções implementadas para executar as operações de validação das expressões infixa (usual), conversão das expressões infixas para posfixas, avaliação do resultado das expressões posfixa. O arquivo também contem implmentações das funções auxiliares como por exemplo substituir chaves e colchetes por parênteses ( replace() ).
